# 茅台酒溯源区块链系统

## 项目概述

本项目是一个基于以太坊区块链技术的茅台酒溯源Demo系统，实现了从生产到销售的全程追踪。通过智能合约记录产品的生产、运输、销售全过程，确保数据的不可篡改性和可追溯性。

## 三、启动系统

### 方式一：使用启动脚本（推荐）

**Windows:**
```cmd
双击 start.bat
```

**Linux:**
```bash
chmod +x start.sh   # 首次运行需要添加执行权限
./start.sh
```

脚本会自动：
1. 启动Hardhat区块链节点
2. 部署智能合约
3. 启动前端开发服务器

### 方式二：手动启动

**步骤1：启动区块链节点（窗口1）**
```bash
cd moutai-trace
npx hardhat node
```

**步骤2：部署合约（窗口2）**
```bash
cd moutai-trace
npx hardhat run scripts/deploy.js --network localhost
```
**重要：复制输出的合约地址！**

**步骤3：启动前端（窗口3）**
```bash
cd moutai-trace/frontend
npm run dev
```

**步骤4：访问应用**

打开浏览器访问：http://localhost:5173

## 四、使用说明

### 4.1 连接合约

1. 打开浏览器访问 http://localhost:5173
2. 点击右上角"设置"按钮
3. 输入部署后的合约地址
4. 点击"连接"

### 4.2 自动模拟（推荐首次使用）

1. 确保当前账户是管理员（账户0）
2. 点击左侧菜单"自动模拟"
3. 点击"开始模拟"按钮

系统会自动：
- 授权12个供应链节点（原料供应商、生产商、批发商、零售商）
- 随机生成3-5个茅台酒产品
- 模拟每个产品从生产到零售的完整流程

### 4.3 产品溯源流程

#### 产品注册（生产厂家操作）
1. 在顶部账户选择器切换到生产厂家账户
2. 点击左侧菜单"产品注册"
3. 填写产品信息并点击"注册产品"

#### 运输管理（物流节点操作）
1. 切换到当前持有产品的账户
2. 点击左侧菜单"运输管理"
3. 选择产品、目标节点、填写地点信息
4. 点击"记录运输"

#### 溯源查询（消费者操作）
1. 点击左侧菜单"溯源查询"
2. 输入产品ID
3. 点击"查询"查看完整运输历史

#### 节点管理（管理员操作）
1. 点击左侧菜单"节点管理"
2. 点击"授权新节点"
3. 填写节点信息并授权

## 五、产品生命周期阶段

| 阶段 | 说明 | 参与者 |
|------|------|--------|
| 原料阶段 | 原料采购与供应 | 原料供应商 |
| 生产阶段 | 酿造与包装 | 生产厂家 |
| 物流阶段 | 仓储与运输 | 批发商/物流中心 |
| 零售阶段 | 销售给消费者 | 零售商 |

## 六、注意事项

1. **本地区块链**: 每次重启Hardhat节点，数据会重置，需要重新部署合约
2. **账户管理**: 不同账户代表不同角色，操作时注意切换正确账户
3. **合约地址**: 部署后需在前端设置中更新合约地址
4. **Gas费用**: 本地测试无真实Gas费用
